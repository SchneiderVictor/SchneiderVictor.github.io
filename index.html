Who am I

I am a student at the University of Toronto and will be graduating this June with a major in Human Biology, a major in Pharmacology and a minor in Computer Science. I invite you to ask me I transitioned to Software Engineering from the Life Sciences, but to keep a long story short: the interest sparked when I was challenged to build a calculator on Minecraft.

What have I done

See below for a brief overview of my published work! Feel free to take a glance at my other, smaller miscellaneous work as well!

See https://github.com/schneidervictor/PCRS_JSLINUX

● An exploratory project to emulate the Linux bash terminal on a web browser. Progress was to be guided and supervised by two professors, however the work was done independently.

● Starting from nothing, I managed to quickly browse and eliminate different options, narrowing down to the most appropriate implementation.

● As a prototype is required, I have taken the initiative to learn JavaScript and the Node.js Express framework.

● This project largely written in JavaScript and C, using emscripten to convert the C code to WebAssembly code.

(Figure 1) The first page served to the browser

This is simply a header with a button that redirects you to the page where the emulator would be. I implemented it this way because the finished product would need a button that could copy some fixed files into a specific folder, based on the user (thus “initializing” that user’s “directory”). At the moment, the code copies original.txt from ./originals/ on the server-side and copies/renames it to copied.txt into ./temp/.
